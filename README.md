## Checklist App
This project is a simple interactive checklist.

## Overview
The app features clean, responsive design, a modern UI, and lets you add, complete, and delete tasks directly in your browser.

## Features
Add new tasks with a simple form

Mark tasks as done (checkbox, with strikethrough style)
Remove tasks with a dedicated button
Modern, clear, and responsive layout

## Technologies Used
HTML5
CSS3 (Flexbox, BEM naming)
JavaScript (ES6, DOM manipulation)
